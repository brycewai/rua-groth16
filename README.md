# rua-groth16

## 基本的Groth16算法

使用rust语言，引入 bls12_381、ff、pairing 三个标准库，实现了最基本的groth16算法：

- 多项式基本运算（+、-、*、/、*=）
- 单位根
- 拉格朗日插值法
- R1CS
- QAP
- Setup
- 生成 Proof
- 验证 Proof
